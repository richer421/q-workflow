# 核心业务能力

描述系统解决的业务问题和核心能力边界。

## 设计原则

- 只描述"做什么"，不描述"怎么做"
- 接口细节由 Swagger 文档维护，此处不重复
- 新增模块时，说明业务场景而非 API 列表

## 能力清单

### Hello World（示例模块）

示例业务场景，用于演示完整的 CRUD 流程和分层架构。

- 创建、查询、更新、删除记录
- 分页列表查询

### MCP Server

独立 MCP Server，供 Claude 调用：

- `q-workflow mcp` — 启动 MCP Server（stdio 模式）
- 工具：
  - `call_api` — 调用 API（action: hello_world.list/get/create/update/delete）
  - `read_logs` — 读取日志文件
